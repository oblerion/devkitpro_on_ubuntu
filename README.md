this script for easy install / setup [devkitpro](https://devkitpro.org/) on ubuntu , i only make this script . 
## how use it
```
sudo chmod +x script.bash
./script.bash
```
## when script run


```
>> devkitpro-pacman installed
...
>> keys installed
```
